Entrar em Outro Domínio

Descrição

É comum que computadores corporativos não permitam a criação de um perfil de Administrador apenas seguindo o procedimento Quebra de Senha. Para realizar os testes, alguns processos adicionais serão necessários.

Soluções

  • Outro Domínio

    Após seguir o procedimento Quebra de Senha:

    1. Clique em Outro Usuário

    2. Clique em Como entrar em outro domínio?

    3. Digite no campo Nome de Usuário o código sugerido nas instruções seguido da palavra Administrador (com a primeira letra maiúscula e as demais minúsculas).

      Exemplo: PE01234/Administrador

    4. Então será criado um perfil de Administrador onde os testes poderão ser realizados.

Obs:

  • No caso de ter-se seguido este procedimento não será possível desativar o Administrador. Nessa situação é permitido finalizar a máquina sem o procedimento documentado em Desativar Administrador.
