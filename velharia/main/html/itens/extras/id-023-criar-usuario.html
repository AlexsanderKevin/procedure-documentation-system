Criar Usuário

Descrição

Em casos como reinstalação do sistema operacional, caso o cliente não tenha solicitado backup, é provável que a máquina chegue à zona de teste sem um usuário. Nesse caso, cabe ao testador criar um usuário apenas para o teste, normalmente com o nome Excluir Usuário.

Existem outras situações em que não haverá reinstalação, porém será necessária a criação de outro usuário. Nesses casos o processo pode ser agilizado com alguns comandos.

Soluções

  • Cortana

    Quando o técnico reinstala o sistema e não cria um usuário, ao iniciar a máquina a primeira coisa a aparecer é a Cortana, a assistente virtual da Microsoft. Cortana auxilia o usuário na criação de um perfil:

    1. Siga as instruções de Cortana até que a assistente recomende conectar-se a uma rede.

    2. Selecione não tenho internet, no canto inferior esquerdo da janela e depois seguir com a configuração limitada no mesmo lugar.

    3. Prossiga com as instruções de Cortana até chegar à criação do perfil do usuário

    4. Atribua ao perfil, o nome Excluir Usuário.

    5. Não atribua uma senha ao perfil.

    6. Prossiga até chegar à área de trabalho do computador.

  • Control Userpasswords2

    No caso de ja haver um perfil criado existe um processo simplificado para criar um outro.

    1. Acesse a barra de comandos com o atalho Win + R e pesquise control userpasswords2.

    2. Na aba Usuários pressione Adicionar... e OK

    3. Seleicone Entrar sem uma conta da Microsoft (não recomendado) e Avançar.

    4. Selecione Conta Local.

    5. Atribua o nome Excluir Usuário e avance sem atribuir uma senha.

    6. O novo perfil estará disponível para testes.

Obs:

  • No LINUX, em caso de reinstalação, o sistema operacional é instalado após os testes, o que requer um processo completamente diferente, sem a criação de usuário.

  • No Chrome OS é possível utilizar o computador entrando como convidado.

  • Após os testes, lembre-se de Excluir Usuário.
