Quebra de Senha

Descrição

Quando o perfil de usuário do cliente vem bloqueado por uma senha é preciso uma maneira de testar a máquina.

No Windows é possível criar um perfil de Administrador no qual será possível realizar os testes. Este perfil, no entanto, não deve estar ativo ao enviarmos a máquina.

Soluções

  • Criando perfil de Administrador

    1. Acesse a BIOS da máquina testada pressionando o botão recovery com o computador desligado, ou pressionando a tecla F2 durante a inicialização.

    2. Verifique se a opção Boot Mode (Modo de Inicialização) está em UEFI na aba Boot (Inicialização).

    3. Na aba Security (segurança) desative a opção Secure Boot (Inicialização Segura).

    4. Conecte o pendrive Quebra Senha na porta USB do equipamento.

    5. Salve a configuração e inicie a máquina pressionando F10 e confirmando.

    6. Acesse o Boot Menu (Menu de Inicialização) pressionando F12 durante a inicialização.

    7. No Boot Menu selecione a opção EFI USB Device (SanDisk).

    8. No terminal que surgirá selecione a opção Lazersoft Live CD.

    9. A máquina inicializará no sistema operacional que está no pendrive.

    10. Selecione Password Recovery e confirme.

    11. Clique em Next, em seguida yes e novamente Next.

    12. Selecione Administrador dentre as opções listadas e depois next.

    13. Clique no botão Reset/Unlock, espere alguns instantes e clique em OK na janelinha que abrir.

    14. Clique em finish e em seguida Reboot para reiniciar a máquina

    15. Lembre-se de reativar o secure boot na BIOS.

    16. Ao inicializar o computador, deve haver um perfil de Administrador siponível para a realização dos testes.

Obs:

  • Lembre-se de que o perfil de Administrador deve ser Desativado antes de finalizar a máquina. Siga o procedimento Desativar Administrador.

  • Não é possível acessar um perfil de Administrador nos sistemas LINUX e Chrome OS.
