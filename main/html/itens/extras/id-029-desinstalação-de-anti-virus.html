Desinstalação de Anti-Virus

Descrição

Os Anti-Virus, normalmente, são softwares muito invasivos, afinal estes precisam verificar a fundo o funcionamento de cada aplicação dentro de um computador. Este pretexto pode acabar sendo utilizado para, no pior dos casos, roubo de dados do usuário ou, no "melhor" dos casos, causar travamento e lentidão exigindo muito da capacidade de processamento da máquina.

Existe, portanto, uma lista de softwares ditos Anti-Virus mas que reconhecidamente prejudicam o funcionamento ou a integridade do computador.

Soluções

  • Anti-Virus a serem desinstalados

    • Santivirus

    • Segurazzo

    • Web Companion

  • Procurando por Anti-Virus

    1. Abra o Painel de Controle

    2. Clique em Desinstalar um programa na seção Programas

    3. Procure pelos softwares citados acima.

    4. Caso haja algum dos Anti-Virus malignos instalado na máquina, informe o técnico responsável.

Obs:

  • É possível que, durante o processo de desinstalação do Anti-Virus, o técnico seja obrigado a reinstalar o sistema operacional, portanto é recomendável procurar pelos Anti-Virus assim que o computador for ligado, antes das atualizações e demais testes.

  • Os sistemas LINUX e Chrome Os não têm suporte aos Anti-Virus listados.
