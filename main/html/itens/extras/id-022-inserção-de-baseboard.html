Iserção do BaseBoard

Descrição

Existe a possibilidade de que, ao se realizar o reparo ou substituição da Placa-Mãe, o BaseBoard também seja deletado.

Para saber se isso realmente aconteceu acesse as Informações do Sistema pressionando Win + R e digitando msinfo32 na barra de pesquisa. Se a informação Produto BaseBoard constar INVALID (inválido), significa que o BaseBoard foi apagado

Soluções

  1. Conecte o pendrive de Inserção de BaseBoard

  2. Execute como Administrador a aplicação Lenovo_BIOS.bat

  3. Siga o passo a passo indicado pela aplicação, informando a unidade do drive e escaneando o modelo da máquina quando necessário.

  4. Conceda à aplicação permissão para reiniciar o computador.

  5. Certifique-se de que o BaseBoard não consta mais como INVALID (inválido) nas Informações do Sistema

Obs:

  • Sistemas como LINUX e Chrome OS não precisam de BaseBoard
