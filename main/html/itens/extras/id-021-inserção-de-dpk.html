Inserção de DPK

Descrição

A DPK é composta de 25 caracteres. Para facilitar a Inserção é possível imprimir a chave em um código de barras para nos permitir escanear o código ao invés de digitá-lo.

Solução

  • Impressão da DPK

    1. Copie o código obtido em Consulta de DPK.

    2. Abra o programa Zebra Designer Pro na sua máquina.

    3. Clique duas vezes no código de barras modelo.

    4. Apague o código escrito e cole a chave obtida.

    5. Imprima a DPK com o comando Ctrl + P e pressionando print.

  • Inserção da DPK

    1. Acesse a BIOS da máquina testada pressionando o botão recovery com o computador desligado, ou pressionando a tecla F2 durante a inicialização.

    2. Uma vez na aba information (informação) da BIOS verifique se a informação QA3 Key ID está zerada e a informação Preinstaled OS License consta No DPK. Isso indica que a BIOS realmente não tem uma DPK gravada.

    3. Verifique se a opção Boot Mode (Modo de Inicialização) está em UEFI na aba Boot (Inicialização).

    4. Na aba Security (segurança) desative a opção Secure Boot (Inicialização Segura).

    5. Conecte o HD externo de inserção de DPK na porta USB do equipamento.

    6. Salve a configuração e inicie a máquina pressionando F10 e confirmando.

    7. Acesse o Boot Menu (Menu de Inicialização) pressionando F12 durante a inicialização.

    8. No Boot Menu selecione a opção EFI USB Device (JMicron)

    9. O computador inicializará pelo Sistema Operacional do HD externo, que contém o software de Inserção de DPK

    10. Uma vez dentro do sistema operacional, antes de qualquer coisa, conecte a máquina à internet.

    11. Acesse a pasta K1_1_0_0_28 localizada na área de trabalho.

    12. Execute a aplicação K1.exe

    13. Faça Login com o Usuário e senha colados no case do HD externo.

    14. Marque a opção Enable Online Mode.

    15. Escaneie ou digite o código da DPK no campo Please log planar's serial number

    16. Pressione o botão Inject.

    17. Espere até aparecer DPK injection done! escrito.

    18. Feche a aplicação pressionando o botão close e desligue a máquina.

    19. Certifique-se de que o equipamento esteja completamente desligado antes de remover o HD externo

    20. Acesse a BIOS novamente e certifique-se de que a DPK foi gravada.

Obs:

  • Retirar o HD externo antes de desligar completamente a máquina pode corromper o sistema operacional instalado no drive.

  • Sistemas como LINUX e Chrome OS não precisam de DPK
