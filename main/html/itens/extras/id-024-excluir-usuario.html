Excluir Usuário

Descrição

É de vital importância que a máquina não seja enviada ao cliente com o perfil de usuário criado em Criar Usuário, portanto siga este procedimento após o teste para garantir que isso não aconteça.

Em caso de reinstalação do sistema operacional, onde o computador deve ser enviado sem nenhum perfil, é necessário, a principio, sair do usuário de testes, pois o Windows não nos permite excluir usuários que estejam em uso no momento.

Soluções

  • Saindo do Usuário de Testes

    1. Acesse a barra de comandos pressionando Win + R e presquise por sysprep.

    2. Execute a aplicação sysprep.exe

    3. Na aplicação, certifique-se de que a Opção de Desligamento esteja em Reinicializar e confirme.

    4. O computador irá reinicializar e quando o sistema for iniciado aparecerá o procedimento de Criar Usuáio com a Cortana.

    5. Pule o procedimento de criação de usuário com o atalho Shift + fn + F3 ou Shift + F3 (caso a primeira tentativa não funcione) e confirme quaisquer procedimentos até que esteja aberto um perfil de Administrador.

    6. É possível que o processo de entrar no perfil de Administrador tenha desativado drivers como o PortCode ou as teclas de atalho, portanto certifique-se de que estejam funcionando, mesmo que seja necessário reinstalá-los.

  • Excluindo o Usuário

    1. Acesse o Painel de Controle pesquisando por control na barra de comandos.

    2. Siga o caminho:

      Contar de Usuário -> Remover Contas de Usuário

    3. Selecione o perfil criado em Criar Usuário.

    4. Pressione Excluir Usuário, Excluir Arquivos e Excluir Conta.

    5. Espere até que o perfil seja excluído.

    6. Caso esteja no peril de Administrador criado no processo acima, abra a aplicação sysprep, configure as Opções de Desligamento para Desligar e confirme.
