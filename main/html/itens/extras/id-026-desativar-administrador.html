Desativar Administrador

Descrição

Os computadores testados, por via de regra, não podem ser enviados ao cliente com o perfil de Administrador ativo.

Soluções

  • Desativa Admin

    1. Acesse o pendrive de testes.

    2. Acesse a pasta Drivers.

    3. Pesquise pelo executável Desativa_Admin.bat

    4. Clique no executável com o botão direito do mouse.

    5. Clique em Executar como Administrador

    6. Conceda a permissão que o sistema pedir.

    7. Pressione qualquer tecla no terminal que for aberto e a máquina reinicializará.

    8. Quando o computador estiver ligado certifique-se de que o perfil de Administrador tenha desaparecido.

Obs:

  • No caso de ter-se seguido o procedimento Entrar em Outro Domínio não será possível desativar o Administrador. Nessa situação é permitido finalizar a máquina sem o procedimento aqui documentado.
